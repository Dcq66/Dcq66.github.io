Java05——数组

2020-09-17

Java 数组：

*数组简介：数组是java中最常见的一种数据结构，用于储存多个数据
*数组的特点：
    - 数组是一种引用数据类型
    - 数组中的多个数据必须类型统一
    - 数组的长度在程序运行期间不可改变

*数组定义：type []arrayName 或 type arrayName[]
           如：int []a 或 int a[]

*数组初始化：在内存中创建一个数组，并且向其中赋予一些默认值
    - 动态初始化（指定长度）：在创建数组时直接指定数组当中的数据元素个数
        - 标准格式：数据类型[] 数组名称=new 数据类型[数组长度];
    - 静态初始化（指定内容）：在创建数组时不直接指定数据个数，而是直接将具体的数据内容进行指定
        - 标准格式：数据类型[] 数组名称=new 数据类型[]{元素1,元素2,....};
        - 省略格式：数据类型[] 数组名称={元素1,元素2,....};
    - 注意事项：
        - 虽然静态初始化没有指定长度，但根据括号中的具体数据内容也可以自动推算出来
        - 动态初始化标准格式可以拆分为两个步骤：
            数据类型[] 数组名称;
            数组名称=new 数据类型[数组长度];
        - 静态初始化标准格式也可以拆分为两个步骤：
            数据类型[] 数组名称;
            数组名称=new 数据类型[]{元素1,元素2,....};
        - 但静态初始化一旦使用省略格式就不能拆分为两个步骤了
* 直接输出数组名称得到的是数组对应的内存地址哈希值
* 访问数组元素的格式：数组名称[索引值]
    - 索引值：一个int数字，代表数组当中元素的编号，若访问数组元素时，访问的索引编号不存在，
                则会发生数组索引越界异常（ArrayIndexOutOfBoundsException）
    - 注意：索引值从0开始，一直到“数组的长度-1”为止
* 数组元素的默认值：
    - 使用动态数组初始化时，其中的数组元素将会自动拥有一个默认值，规则如下：
        - 整数类型则默认为0
        - 浮点类型则默认为0.0
        - 字符类型则默认为'\u0000'
        - 如果是布尔类型则默认为false
        - 如果是引用类型则默认为null
    - 静态初始化其实也有默认值，只不过系统马上将默认值自动替换成了大括号当中具体的数据了
* 空指针异常（NullPointerException）
    - 所有的引用类型变量都可以赋值为一个null值，这代表其中什么都没有，
        数组必须进行new初始化才能试用期中的元素，若只是赋值了一个null，没有进行new创建
            那么将会发生空指针异常
* 获取数组的长度：数组名称.Length
