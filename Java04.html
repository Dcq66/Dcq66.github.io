Java数组：

*数组简介：数组是java中最常见的一种数据结构，用于储存多个数据

*数组定义：type []arrayName 或 type arrayName[]
           如：int []a 或 int a[]

*数组初始化：
    > 静态数组初始化：arrayName=new type[](....)
                      如 int []a=new int[](1,2,3)
    > 动态数组初始化：arrayName=new type[length]
                      如 int []a=new int[3]
