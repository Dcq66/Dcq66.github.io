Html01

2020-09-16

Html标签：

* html对于网页的编写内容都要存在于标签内(标签一般都是以一对的形式出现)，基本标签如：
    <html> </html> : 根标签，html编写网页的所有内容都必须在根标签内，一个页面中有且只有一个根标签
    <head> </head> : 设置网页头部信息，在此标签内编写的内容在网页中都不会显示出来，主要作用是帮助浏览器去解析网页
    <body> </body> : 设置网页的主体部分，网页中所有可见的内容都应写在此标签中
    <title> <title> : 网页的标题编写在此标签中，默认显示在浏览器的标题栏中，搜索引擎在检索页面时会首先检索此标签
                      中的内容，他是页面中对于搜索引擎最重要的内容，会影响到页面在搜索引擎中的排名
    <h1> </h1> : 网页内容的一级标题，多级标题以此类推，一共有六级标题
    <!-- --> : 注释在此标签的空格处编写，注释在网页中不会显示，但在源码中会显示出来
* html标签的属性：
    > 可以通过属性来设置标签如何处理标签中的内容
    > 只能在开始标签内设置属性
    > 属性实际上就是一个名值对的结构
    > 属性名="属性值"，一个开始标签中可以设置多个属性，属性之间需要用空格间隔开来

Java02——数据类型和运算符

2020-09-15

Java 数据类型:

分为基本数据类型和引用数据类型：
* 基本数据类型
    > 数值型
        > 整数类型(byte,short,int,long)
        > 浮点类型(float,double) 其中小数默认为double类型，所以当表达一个float类型时，必须加一个f,如float a=1.1f;
    > 字符型(char)
    > 布尔型(boolean) 布尔类型只有true(真)和false(假)两种，主要用于逻辑判断
* 引用数据类型
    > 类(class)
    > 接口(interface)
    > 数组

基本类型的转换：
* 自动类型转换
    > 条件1.转换前后的数据类型兼容
    > 条件2.转换后的数据类型的表示范围要比转换前的大

常用的 DOS(cmd)命令：2w22s

  • exit 退出当前 DOS(cmd)命令窗口
  • cls 清屏
  • dir 列出当前目录下所有的子文件/子目录
  • cd 路径 转入想要进入的路径之下
    路径包括绝对路径和相对路径
    绝对路径：表示该路径从某个磁盘的盘符下作为出发点的路径 如 C:\Users\Lenovo>
    相对路径：表示该路径从当前所在的路径下作为出发点的路径 如 blog
  • cd .. 回到上级目录
  • cd / 直接回到根目录
  • 切换盘符：c:
